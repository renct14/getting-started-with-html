    The web

    The web are numerous computers communicating with each other. Through your browser (e.g. Chrome or Firefox) you can send HTTP requests to servers through the internet. That server will then fetch and return the appropriate HTML documents, which your web browser will interpret and display back to you. The web mainly contains HTML documents, but can also have many other things including images, videos, PDFs, etc.

    HTML

    HTML stands for Hyper Text Markup Language. HTML is made up of:

  • text content - what you see
  • markup - what it looks like
  • references to other documents - e.g. images and videos
  • links to other pages

    • A HTML element is made up of an opening tag followed by the content and a closing tag. A void tag is a type of tag that has no content.
    Another thing to note is that any whitespace in HTML will automatically turn into a single space.
